# /buildflow-start

> **This command has been renamed to `/buildflow-start-epic`.**

`/buildflow-start-epic` is the current command for beginning or resuming a project. It includes vision capture, codebase drift detection, epic history loading, and session setup.

## What to run instead

```
/buildflow-start-epic
```

Everything works the same — the name change reflects the shift from numbered phases to named epics (`1-auth`, `2-payments`, etc.).
